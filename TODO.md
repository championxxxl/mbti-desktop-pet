## 🤝 如何开始贡献 / How to Start Contributing

### 第1步：选择任务
从上面的任务列表中选择一个适合你的任务

### 第2步：创建分支
```bash
git checkout -b feature/任务名称
# 例如: git checkout -b feature/fix-intent-recognition
```

### 第3步：开发
按照任务要求进行开发，记得：
- 遵循现有代码风格
- 添加必要的注释
- 写清晰的commit信息

### 第4步：测试
```bash
# 运行demo测试
python demo.py

# 运行单元测试（如果有）
pytest tests/

# 手动测试你的更改
```

### 第5步：提交PR
```bash
git add .
git commit -m "完成任务：描述你的更改"
git push origin feature/任务名称
```

然后在GitHub上创建Pull Request
